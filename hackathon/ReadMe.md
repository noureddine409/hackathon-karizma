### Prerequisites

- Java 17
- Maven

### Installation

1. Clone the repository:
  ```bash
  git clone https://github.com/noureddine409/-sink.git
  ```
2. Navigate to the project directory:
  ```bash
  cd notification-sink
  ```
3. Build the project:
  ```bash
  mvn clean install
  ```
4. Run the application:
  ```bash
  mvn spring-boot:run
  ```
